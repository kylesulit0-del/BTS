**Create `src/hooks/useVideoAutoplay.ts`:**

Module-level state for one-at-a-time enforcement:
- `currentlyPlaying: { iframe: HTMLIFrameElement; videoType: VideoType } | null` -- tracks the single active video
- Export `useVideoAutoplay(containerRef, iframeRef, videoType, onVisibilityChange?)` hook

Hook logic:
1. Create IntersectionObserver with `threshold: 0.5` (per research recommendation)
2. When entry `isIntersecting` with ratio >= 0.5:
   - If another video is playing, pause it via `sendCommand(currentlyPlaying.iframe, currentlyPlaying.videoType, 'pause')`
   - Play this video via `sendCommand(iframe, videoType, 'play')`
   - Update `currentlyPlaying`
3. When entry leaves viewport and this is the currently playing video:
   - Pause via `sendCommand`
   - Set `currentlyPlaying = null`
4. Cleanup: disconnect observer on unmount

`sendCommand` helper function:
- YouTube: `iframe.contentWindow?.postMessage(JSON.stringify({ event: 'command', func: funcName, args: [] }), 'https://www.youtube.com')`
  - `funcName`: `'playVideo'`, `'pauseVideo'`, `'mute'`, `'unMute'`
- TikTok: `iframe.contentWindow?.postMessage({ type: command, 'x-tiktok-player': true }, 'https://www.tiktok.com')`
  - `command`: `'play'`, `'pause'`, `'mute'`, `'unmute'`

Also export a `sendMuteCommand(iframe, videoType, mute: boolean)` for the unmute-on-tap feature.

**Create `src/components/VideoEmbed.tsx`:**

Props: `{ videoType: VideoType; videoId: string; title: string; thumbnail?: string }`

Component logic:
1. Refs: `containerRef` (div), `iframeRef` (iframe)
2. State: `loaded` (boolean, false initially), `muted` (boolean, true initially -- browser requires muted autoplay), `embedError` (boolean, for TikTok fallback)
3. Call `useVideoAutoplay(containerRef, iframeRef, videoType)`
4. Build iframe src:
   - YouTube Shorts: `https://www.youtube.com/embed/${videoId}?enablejsapi=1&autoplay=0&mute=1&loop=1&playlist=${videoId}&playsinline=1&rel=0&controls=1`
   - TikTok: `https://www.tiktok.com/player/v1/${videoId}?autoplay=0&loop=1&controls=1&progress_bar=1&play_button=1&volume_control=1&music_info=0&description=0&rel=0`
5. Handle `onLoad`: set `loaded = true`
6. Handle `onError`: set `embedError = true` (for TikTok fallback)
7. Mute indicator: show a small mute icon overlay when `muted === true`. On click/tap of the video container, call `sendMuteCommand` to unmute and set `muted = false`.

Render:
```
<div ref={containerRef} className="video-embed" style={{ aspectRatio: '9 / 16' }}>
  {/* Loading placeholder: thumbnail + spinner, visible until iframe loaded */}
  {!loaded && (
    <div className="video-embed-placeholder">
      {thumbnail && <img src={thumbnail} alt="" />}
      <div className="video-embed-spinner" />
    </div>
  )}
  {/* TikTok fallback when embed fails */}
  {embedError && videoType === 'tiktok' ? (
    <a href={`https://www.tiktok.com/@/video/${videoId}`} target="_blank" rel="noopener noreferrer" className="video-embed-fallback">
      {thumbnail && <img src={thumbnail} alt={title} />}
      <span className="video-embed-fallback-label">Open in TikTok</span>
    </a>
  ) : (
    <iframe ref={iframeRef} src={src} title={title}
      allow="autoplay; encrypted-media"
      allowFullScreen
      onLoad={() => setLoaded(true)}
      onError={() => setEmbedError(true)}
      style={{ opacity: loaded ? 1 : 0, position: 'absolute', inset: 0, width: '100%', height: '100%', border: 'none' }}
    />
  )}
  {/* Mute indicator */}
  {loaded && muted && (
    <button className="video-embed-mute-btn" onClick={handleUnmute} aria-label="Tap to unmute">
      {/* Inline SVG muted speaker icon */}
    </button>
  )}
</div>
```
