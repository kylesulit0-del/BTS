   - On first load: check if URL params `?sort=`, `?source=`, `?type=` exist. If so, migrate them to localStorage initial state and clear URL params (smooth transition from PERF-03). Use `window.history.replaceState` to clear params without navigation.

   - `useEffect` to save state to localStorage on every state change (debounce not needed -- writes are infrequent)

5. Keep `SortMode` type export unchanged.
