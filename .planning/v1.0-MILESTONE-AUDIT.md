- Assigned to Phase 4, not in v1.0 scope

### Partial Requirements (integration gaps)

**SRC-04**: News filter tab is non-functional
- `FeedFilter.tsx` line 12: `value: "news"` does not match RSS fetcher `source: "rss"`
- Clicking "News" tab returns zero items
- **Fix:** Change `value: "news"` to `value: "rss"` in FeedFilter.tsx

**EMBED-02**: TikTok short URL embeds broken
- `vm.tiktok.com` / `vt.tiktok.com` short URLs detected, but the short path (e.g. `ZS1ABc`) is passed as videoId to TikTok's `player/v1` endpoint which expects numeric IDs
- Full TikTok URLs work correctly
- **Fix:** In VideoEmbed, detect `isShortUrl` and render fallback link card instead of broken iframe
